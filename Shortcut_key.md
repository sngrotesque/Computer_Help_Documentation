# 键盘快捷键

> Linux与Mac OS用户请注意，部分快捷键仅在Windows系统中证实。

```text
此处需要说明一些东西，此快捷键文档不一定完全全面。
并且有些计算机的基础操作并不能很好的被说出来，
但可以在此处给你列出一些。

在选择文件或内容时
    使用Ctrl键与方向键或鼠标协同可以更好的选择内容。
    使用Shift键与方向键或鼠标协同可以更好的选择内容。
    当然，除了键盘和鼠标以外你还可以协同6大功能键
        insert home PageUP delete end PageDown

最后祝你使用计算机的过程愉快！
```

<span id="Single_key">单键</span>
```text
ESC             返回，退出，离开
F1              帮助键
F2              重命名选中内容
F3              搜索窗口(类似Ctrl F)
F4              编辑网址/地址栏
F5              刷新页面(如网页，资源管理器)
F6              定位到地址栏，在浏览器中为将你的光标移动到标签列
F7              在Windows中没有任何作用，在Word中可以进行拼字检查
F8              在启动电脑时可以显示启动菜单，在Media Player可以降低音量
F9              在Windows中没有作用，在Media Player可以用来调高音量
F10             相当于Alt快捷键
F11             全屏页面(游戏，网页，视频等)
F12             在Word中按下它可以快速另存新档
Shift           在中文输入法中将语言在中英切换
Tab             制表符或页面定位器
Ctrl            单独使用无意义
Alt             调出菜单(功能极为强大)
Fn              单独使用无意义
Win             打开开始菜单

Print Screen    全屏截图(部分键盘为PrtSc)
Scroll Lock     滚动锁定(部分键盘为ScrLK)
Pause Break     中断暂停(部分键盘为Pause)
backspace       退格键(在键盘上通常表示为向左箭头)
Insert          切换覆盖模式(部分键盘为INS)
Home            定位至行头
Page UP         向上翻页(部分键盘为PgUP)
DELETE          删除指定内容(部分键盘为DEL)
End             定位至行尾
Page Down       向下翻页(部分键盘为PgDN)
Numeric Lock    数字键盘锁定(部分键盘为NUM Lock或NUM)
Caps Lock       大写锁定(部分键盘为CapsLK)

记事簿图标      与鼠标右键功能相同
```

<span id="Combination_key">组合键</a>
```text
Ctrl A              全选页面(Control All)
Ctrl B              选中的字体加粗(通常用于Word等软件)
Ctrl C              复制选中的内容
Ctrl D              删除(浏览器中通常作为收藏功能键)
Ctrl E              通常用于软件Excel等软件的功能键
Ctrl F              查找指定内容
Ctrl G              定位指定内容(类似Ctrl F)
Ctrl H              替换指定内容为新内容
Ctrl I              通常用于(Word, Excel)等软件的字体倾斜体
Ctrl J              在Word中是两端对齐，在PS中为复制图层
Ctrl K              在Word中为添加超链接的功能键
Ctrl L              在Word中为左对齐功能键
Ctrl M              在PS中为曲线功能键
Ctrl N              新建指定内容(如工程，网页，窗口)
Ctrl O              打开指定内容，在PS中为把图层缩放到整体可以显示
Ctrl P              打印指定内容
Ctrl Q              在Word中为清除选择文本的字体格式
Ctrl R              在浏览器中为重载网页，在Word中为向右填充
Ctrl S              保存指定内容(Control Save)
Ctrl T              在PS中为自由变换功能键，在Excel中为插入表格
Ctrl U              在PS中为调整色相/饱和度的功能键
Ctrl V              粘贴刚才复制的内容至指定位置
Ctrl W              关闭程序，在浏览器中为关闭当前页面
Ctrl X              剪切指定内容(类似Ctrl C，但删除原数据)
Ctrl Y              还原通过Ctrl Z功能键撤回的内容，在PS中为转换图层模式RGB与CMYK的
Ctrl Z              撤回操作

Ctrl ESC            打开开始菜单
Ctrl Tab            程序窗口切换
Ctrl Space          中英输入切换
Ctrl Lift           将光标向左按照单字头部移动
Ctrl Right          将光标向右按照单字尾部移动
Ctrl Up             查看上方视图内容
Ctrl Down           查看下方视图内容
Ctrl Home           移动至窗口顶部
Ctrl End            移动至窗口底部
Ctrl PageUP         向左切换标签页
Ctrl PageDown       向右切换标签页

Ctrl Alt Delete     启动Windows安全选项
Ctrl Alt ESC        启动任务管理器
Ctrl Shift Tab      在浏览器中为切换至上一个标签页
Ctrl Shift N        新建文件夹/窗口，浏览器中为新建隐私窗口

Win Tab             显示所有已打开的窗口
Win A               打开系统通知窗口
Win B               定位到任务栏的"隐藏"
Win C               调出Cortana(小娜)
Win D               最小化所有窗口，再次执行恢复所有窗口
Win E               打开资源管理器
Win F               打开反馈中心
Win G               打开游戏模式(Windows系统)
Win H               启动听写功能
Win I               打开Windows设置
Win J               无意义
Win K               Windows设置 -> 设备 -> 连接
Win L               锁屏
Win M               最小化所有窗口
Win N               无意义
Win O               无意义
Win P               改变显示器投影类型
Win Q               打开搜索
Win R               运行
Win S               打开搜索
Win T               将光标定位到任务栏
Win U               Windows设置 -> 轻松使用
Win V               打开剪贴板
Win W               启动手写笔
Win X               超级菜单
Win Y               无意义
Win Z               无意义

Alt Tab             切换应用窗口选项卡
Alt ESC             功能与Alt Tab相同但顺序相反
Alt Shift           与Win Space相同
Alt F               显示程序菜单
Alt F4              关闭应用/关机菜单
Alt Enter           查看文件属性
Alt L/R/U/D         (Lift, Right, Up, Down)在不同软件中功能不同(很有趣)
                    Alt Lift在资源管理器中是返回上一级目录
                    Alt Up与Alt Lift相同
Alt D               选中地址栏


```
